# Merika's FH5 修改器（简体中文版）

## 功能特性

- **开源软件**：基于 GNU 通用公共许可证 v3.0 完全开源。

- **车辆作弊**
  - 速度修改、刹车修改、跳跃修改、轮速调整和瞬间停止
  - 重力修改、航点传送、冻结AI、无水阻力和穿墙

- **解锁作弊**
  - 即时获得金币、经验值、技能点、抽奖、赛季点数和系列点数

- **调校作弊**
  - 直接编辑空气动力、外倾角、前束、防倾杆、悬挂高度和限制值

- **友好的用户界面**
  - 便捷的修改/作弊开关和滑块，快速调整

- **社区支持**
  - Discord集成和故障排除链接
